# Algoritmo_de_enjambre_PSO
en este proyecto, implementamos el PSO, con la finalidad de resolver un problema de ubicaciones de sensores para la maxima eficiencia de cobertura, teniendo en cuenta varios factores como la humedad, salinidad, etc. necesitamos guardar el archivo datos.csv en la misma carpeta que el main.py para poder acceder a los datos del excel con coordenadas.
